## 介绍

定位是一个 Minecraft 工业模组，虽然目前只有简单的元素，但是却画着很大的饼。~~（真的很大）~~

模组提供了一套 API，提供了一套能量系统以及一套动能系统可供附属模组接入。

本模组提供的两套系统十分贴近现实，拥有转速和功率等设定。

同时也可以为玩家带来令人惊叹的，对于工业之美的体验，足够贴近真实。

未来将作为《三体工业》的前置 Mod。
