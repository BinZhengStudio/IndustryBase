# 模组介绍
定位是一个 Minecraft 工业模组，虽然目前只有简单的元素，但是却画着很大的饼。~~（真的很大）~~

模组提供了一套 API，提供了一套能量系统可供附属模组接入。

未来将作为《三体工业》的前置 Mod。
